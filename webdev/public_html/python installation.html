Jupyter Notebook Installation

Note:-
Install Python 3 or Higher Don't Go For Python as suggested in the video



Windows and Mac Users

  • Follow up the video to install


    • Linux User

  • 1.Follow up the video to Download Anaconda3-5.2.0-Linux-x86_64.sh
  • 2.After download is complete drag and move the dowloaded file into home directory
  • 3.Open terminal (ctrl+alt+t) and type bash Anaconda3-5.2.0-Linux-x86_64.sh welcome message will appear
  • 4.Simply Hit Enter or Type Yes Where It Ask
  • 5.Once Anaconda Installation Is Successful type sudo apt install python-pip
  • 6.Once pip is installed type sudo pip install jupyter
  • 7.After jupyter installation is complete you can run jupyter notebook by typing jupyter notebook
  • 8.Your Browser Will Popup Automatically With Jupyter Notebook Opened
